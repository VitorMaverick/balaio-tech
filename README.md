# Balaio.tech — Blog Pessoal

Blog pessoal de um estudante de tecnologia compartilhando o que está aprendendo.

## Tecnologias
- HTML5, CSS3, JavaScript puro
- Swiper.js (slider)
- Typed.js (efeito de digitação)
- Google Fonts (Fira Code + Inter)

## Estrutura
- `index.html` — página principal
- `assets/` — CSS, JS, imagens
- `technologies/` — artigos organizados por categoria
